## Пункт 1 (ACTION). Труба обсадная и труба ПНД: решение пользователя, готово к внедрению

## Пункт 5 (проверено, действий не требуется). Труба в геотекстиле: `drenazhnaya-truba` и `geotekstil-dlya-drenazha`

## Ключи-шум, исключённые из выводов

Как вы и предупреждали, в блоке «фильтр для воды из скважины» и в блоке гидроаккумулятора нашлись строки не по теме страницы, похоже на коллизию омонима «фильтр» (то ли фильтр очистки воды, то ли геотекстильный фильтр-обмотка на дренажной трубе) и смежную технику (дренажный насос вместо скважинного). В рекомендации не тянул:

- «дренажная труба 110 в фильтре» (119 просмотров): про геотекстильную обмотку трубы, не про водоочистку;
- «дренажная труба 110 в фильтре геотекстиль» (851): то же, и с высоким объёмом; по смыслу это кандидат для `drenazhnaya-truba`/`geotekstil-dlya-drenazha`, не для `filtr-dlya-vody-iz-skvazhiny`;
- «труба дренажная с фильтром» (43): аналогично;
- «фильтр для дренажного насоса» (112): про насос, не про очистку воды из скважины;
- «дренажный фильтрационный колодец» (303): про поглотительный колодец, тема уже покрыта в `drenazhnyy-kolodec`;
- «фильтр для бассейна из дренажного насоса» (390): про бассейновое оборудование, к сайту не относится вообще;
- «реле давления для дренажного насоса» (33, блок гидроаккумулятора): про дренажный (сливной, фекальный) насос, не про скважинный насос с гидроаккумулятором.

Итого из 3766 просмотров блока «фильтр для воды из скважины» примерно 1818 (около половины) - шум не по теме страницы; реальный релевантный объём для `filtr-dlya-vody-iz-skvazhiny` - около 1948 просмотров. Полезно знать при будущей приоритизации контента, но на решения по каннибализации это не повлияло.
